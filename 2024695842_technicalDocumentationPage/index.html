Introduction

HTML stands for HyperText Markup Language.

It is the standard language for creating web pages.

<html></html>
  • HTML creates page structure.
HTML Basics

HTML uses tags to define elements.

Tags are enclosed in angle brackets.

<h1>Hello</h1>
  • Opening tag
  • Closing tag
HTML Elements

An element consists of a start tag and an end tag.

Elements can contain text and other elements.

<p>Paragraph</p>
  • Paragraph
HTML Attributes

Attributes provide additional information.

They are written inside the opening tag.

<a href="">Link</a>
  • href
HTML Forms

Forms are used to collect user input.

Forms contain input elements.

<input type="text">
  • Text input
  • Password input
